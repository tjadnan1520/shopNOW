# 🛒 ShopNOW

ShopNOW is a simple **front-end e-commerce web application** built using **HTML, CSS, and JavaScript**.  
It allows users to browse products, explore categories, and view product details through a clean and user-friendly interface.

This project demonstrates fundamental **web development concepts** and a basic **e-commerce UI design**.

---



## 📌 Features

- 🛍️ Browse products from different categories
- 🔎 Search for products
- 📄 Individual product details page
- 🧾 Blog section
- ℹ️ About page
- 📞 Contact page
- 💻 Responsive and simple UI design

---

## 🧑‍💻 Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Interactivity

## ⚙️ Installation

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/tjadnan1520/shopNOW.git
```

### 2. Go to the project directory

```
cd shopNOW
```

### 3. Open the project

Open **index.html** in your browser.
